Welcome

Base Frame makes an installer for your AutoHotkey application! It has following features:

  • Make an installer.

    • Normal and / or portable installation

    • Multiple installer languages

    • Shows a license during installation

    • Can show a changelog during installation

    • Do updates

  • Update checker (optional)

    • Does automatic or manual checks for update and notifies user.

    • Downloads the update and starts an update.

    • Make your app automatically check for updates on each run
