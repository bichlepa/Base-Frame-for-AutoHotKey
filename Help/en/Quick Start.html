Quick start

Base Frame is simple! That's all you need to do to create an installer:

  • Open Base Frame and click on Open.

  • Navigate to your application and select the main script.

  • A window opens. Check and modify the fields.

  • Now you see this window:

    On the right-hand side you see which files and folders will be included to the installer. Add more files if you want. You can also add some other scripts which will be compiled and added to the installer.

  • Click on Generate installation file.

  • After the generation process you can find the installer.
